Removes commas from arrays and objects, using newlines for separation instead.

In MAML, commas are optional when items are on separate lines. This intention removes all commas from an array or object and ensures each item is placed on its own line, creating a more readable format for multi-item containers.

Before (array):

items: [1, 2, 3]

After (array):

items: [
  1
  2
  3
]

Before (object):

{name: "John", age: 30, city: "NYC"}

After (object):

{
  name: "John"
  age: 30
  city: "NYC"
}

Note: This intention automatically adds newlines between items before removing commas to ensure the result remains valid MAML syntax (commas are required on the same line, but optional across lines).
