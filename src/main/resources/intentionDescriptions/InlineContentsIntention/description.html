Recursively inlines/flattens container contents onto a single line, adding commas as needed.

This intention takes arrays and objects that span multiple lines and collapses them to a single line, adding commas between items where required. It works recursively, so nested containers are also inlined.

Before (array):

items: [
  1
  2
  3
]

After (array):

items: [1, 2, 3]

Before (nested object):

config: {
  server: {
    host: "localhost"
    port: 8080
  }
  debug: true
}

After (nested object):

config: {server: {host: "localhost", port: 8080}, debug: true}

Special behavior: When invoked on a key (rather than inside a container), this intention will inline the value's contents if the value is an array or object.

Note: This is the opposite of "Remove commas" which breaks items onto separate lines.
