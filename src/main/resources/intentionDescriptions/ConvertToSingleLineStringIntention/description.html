Converts a multiline string to a single-line string format.

In MAML, single-line strings are delimited with regular quotes (") and require special characters to be escaped. This intention escapes newlines, tabs, quotes, backslashes, and other special characters, converting them to their escape sequences.

Before:

message: """
Hello
World	Test"""

After:

message: "Hello\nWorld\tTest"

All whitespace within the string content is preserved. Special characters are escaped as: \\ (backslash), \" (quote), \n (newline), \r (carriage return), \t (tab), and \b (backspace).
