Converts a single-line string to a multiline string format.

In MAML, multiline strings are delimited with triple quotes (""") and allow you to include newlines and special characters without escaping them. This intention unescapes common escape sequences like \n, \t, and \", converting them to their literal characters.

Before:

message: "Hello\nWorld\tTest"

After:

message: """
Hello
World	Test"""

The content always starts on a new line after the opening """. The closing """ is placed on the same line as the last content unless the string ends with a newline character.
